title: wsgiServer API

<h2>wsgiServer</h2>
This module provides a multithreaded WSGI server implementation.

<h3>WSGIServer (serverAddress, applicationMap)</h3>
The serverAddress parameter is a (address, port) tuple that specifies the interface and port number that the server should listen on.  The applicationMap parameter is a dictionary object that contains a map between URL paths and WSGI applications.

The class is a sub-class of both SocketServer.ThreadingMixIn and BaseHTTPServer.HTTPServer.

<h4>Example</h4>
<pre><code>server = wsgiServer.WSGIServer (('localhost', 1088), wsgiCallable)
server.serve_forever()
</code></pre>
